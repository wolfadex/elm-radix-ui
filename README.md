# Elm Radix UI

A port of [Radix UI](https://www.radix-ui.com/) to Elm to make for quick prototyping and easily extending your prototype into a long term project.

## Supported Components

### Layout

-   [x] Box
-   [x] Flex
-   [x] Grid
-   [x] Container
-   [x] Section

### Typography

### Getting Started

Clone this template, then run

```sh
npm install
npm run dev
```
